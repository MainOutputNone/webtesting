Introducão:

Esta é minha pagina '00'. Ela contém elementos básicos da linguagem de marcação HTML5, assim como este pequeno texto de apresentação que estou escrevendo que usa o elemento de parágrafo 'p' para formatar um texto em um parágrafo.

Aqui estão escritos estes dois parágrafos iniciais. Usei o editor vim, em uma nova variação de sí chamada neovim, como meu editor de código para este projeto, e junto a distribuição LunarVim. Todo o projeto está sendo desenvolvido usando como ferramenta de versionamento o git.

Objetivo da página:

O óbvio é a apresentação dos conhecimentos básicos de elementos HTML5, e de ínicio fazer a alteração de como o conteúdo é mostrado usando "stylesheet" em CSS3 para formatar essa mudança.
